# 🌍 War Estruturado em C

Projeto desenvolvido para a disciplina de Engenharia de Software, com o objetivo de praticar o uso de **structs**, **arrays** e **entrada/saída de dados** na linguagem C.

---

## 📌 Objetivo

Criar um sistema simples que:

- Permite o cadastro de 5 territórios
- Armazena nome, cor do exército e número de tropas
- Exibe todos os territórios cadastrados ao final

---

## 🧠 Conceitos utilizados

- Structs (estruturas de dados compostas)
- Vetores (arrays)
- Laços de repetição (`for`)
- Entrada de dados com `scanf` e `fgets`
- Manipulação de strings com `string.h`
